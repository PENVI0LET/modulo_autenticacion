Introduction

Base URL: {{ config("app.url") }}

Bienvenido a la documentación de nuestra API. Aquí podrás encontrar todos los endpoints disponibles, junto con ejemplos de uso para integrarte fácilmente con nuestra plataforma.

Para interactuar con los endpoints, puedes usar el botón "Try it out" a continuación de cada uno.

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Para autenticarte, utiliza el token Bearer en el header Authorization.

Endpoints

Registra un nuevo usuario en el sistema.

Este método valida los datos del usuario, crea un nuevo usuario en la base de datos y genera una respuesta con la información del usuario creado.

Example request:
curl --request POST \
    "{{ config("app.url") }}/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"distinctio\",
    \"email\": \"rafaela45@example.net\",
    \"password\": \"\'fiD~;]\",
    \"password_confirmation\": \"doloribus\"
}"

const url = new URL(
    "{{ config("app.url") }}/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "distinctio",
    "email": "rafaela45@example.net",
    "password": "'fiD~;]",
    "password_confirmation": "doloribus"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = '{{ config("app.url") }}/api/register';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'distinctio',
            'email' => 'rafaela45@example.net',
            'password' => '\'fiD~;]',
            'password_confirmation' => 'doloribus',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Inicia sesión de un usuario con email y contraseña.

Si las credenciales son correctas, genera un token JWT para la sesión del usuario.

Example request:
curl --request POST \
    "{{ config("app.url") }}/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"camilla.kub@example.org\",
    \"password\": \"0_}&P\\/0\\\"\\\\oanq1Q9#H\"
}"

const url = new URL(
    "{{ config("app.url") }}/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "camilla.kub@example.org",
    "password": "0_}&P\/0\"\\oanq1Q9#H"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = '{{ config("app.url") }}/api/login';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'camilla.kub@example.org',
            'password' => '0_}&P/0"\\oanq1Q9#H',

);
$body = $response->getBody();
print_r(json_decode((string) $body));
